# BillionairesRegistry
DLL CRUD registry system

## Billionaires Registry System

Ένα πρόγραμμα γραμμένο σε C για την καταχώρηση, διαχείριση, αναζήτηση και ταξινόμηση δεδομένων δισεκατομμυριούχων, χρησιμοποιώντας κυκλική διπλά συνδεδεμένη λίστα (circular doubly linked list).

## Περιγραφή

Το πρόγραμμα επιτρέπει στον χρήστη να:

Προσθέτει νέους δισεκατομμυριούχους

Διαγράφει την πιο πρόσφατη ή την παλαιότερη καταχώρηση

Φορτώνει δεδομένα από αρχείο

Αποθηκεύει τα δεδομένα σε αρχείο

Εκτελεί αναζητήσεις με βάση όνομα, χώρα, δραστηριότητα

Φιλτράρει με βάση πλούτο, ημερομηνία γέννησης ή ώρα γέννησης

Ταξινομεί την λίστα (αλφαβητικά, κατά ηλικία, κατά καθαρή αξία)

Υπολογίζει το συνολικό net worth

Η δομή δεδομένων είναι κυκλική ώστε να επιτρέπει αμφίδρομη πλοήγηση και εύκολη εισαγωγή/διαγραφή.

## Δομές
Bdate

Αποθηκεύει ημερομηνία γέννησης.

Btime

Αποθηκεύει ώρα γγέννησης.

Billionaire

Περιέχει όλα τα στοιχεία ενός δισεκατομμυριούχου, όπως:

Όνομα

Πόλη/Χώρα

Δραστηριότητα

Χώρα προέλευσης

Φύλο

Ημερομηνία & ώρα γέννησης

Καθαρή αξία

Αυτόματα παραγόμενο account number

## Βασικά Features
# Προσθήκη δισεκατομμυριούχου
newbil όνομα;πόλη-χώρα;δραστηριότητα;χώρα-προέλευσης;φύλο;ΗΗ/ΜΜ/ΕΕΕΕ ΩΩ:ΛΛ;networth

# Διαγραφή
deleteN   // διαγράφει την πιο πρόσφατη εγγραφή  
deleteO   // διαγράφει την πιο παλιά

# Αποθήκευση / Φόρτωση
save filename.txt
load 10 filename.txt

# Αναζήτηση
findN όνομα
findL χώρα ή πόλη
findS δραστηριότητα
findN *   // εκτύπωση όλων

# Φίλτρα ανά σύγκριση
print+ value     // μεγαλύτερη καθαρή αξία
print- value     // μικρότερη
print= value     // ίση

print+ ΗΗ/ΜΜ/ΕΕΕΕ     // μεταγενέστερη ημερομηνία
print-               // προγενέστερη
print+ ΩΩ:ΛΛ         // μεταγενέστερη ώρα

# Ταξινόμηση
sortA   // αλφαβητικά
sortT   // κατά ηλικία (από μεγαλύτερο σε μικρότερο)
sortW   // κατά net worth

# Άλλες λειτουργίες
sum   // εμφάνιση συνολικού net worth
exit  // έξοδος

## Σημειώσεις

Το πρόγραμμα ελέγχει για διπλότυπες εγγραφές.

Αν κάποιο πεδίο είναι '?', αυτόματα γίνεται 'Unknown'.

Το account number δημιουργείται με αλγόριθμο τύπου Luhn.

Το πεδίο φύλου δέχεται: M, F, NB, TG, Q, O.
